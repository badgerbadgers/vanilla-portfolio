AppointmentApp

This app gets randomized API data and display that info as an appointment.

An app that gets data and will display that data as a random number of appointments.

Each user displayed on the app has a time for that appointment and a delete button when the meeting is done.

There are two tabs that allow for navigation, the 'current meetings' tab shows active meetings. The 'deleted meetings' tab shows any deleted meetings.

There is also a counter that keeps track of the number of meetings the user has for the day. Built using React hooks and deployed using Netlify.
