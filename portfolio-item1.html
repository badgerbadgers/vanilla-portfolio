Tim'sTodo List

A todo list that get API data and lets a user add, edit and delete a todo!

This todo list is gets data from Airtable (an external API source) and using React hook displays that data in the app.

The user is able to add, delete and edit a todo by typing in the input field, clicking the trash or pencil icon. These features also update the Airtable API by using the browser's built in fetch method as different requests types (POST, UPDATE, DELETE).

A checkbox let's a user add a checkmark next to the todo and deleting or adding a todo will update a counter at the bottom of app. The app is fully responsive on all media devices and currently deployed using Netlify.
